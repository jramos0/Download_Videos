# Video and Audio Conversion Scripts

This repository contains scripts for downloading YouTube videos and converting them to audio, as well as for batch-converting `.mp4` videos to `.mp3` files. Follow the setup instructions below to use each script.

## Requirements

- Python 3.x
- Required libraries:
  - `yt_dlp` for downloading videos from YouTube.
  - `moviepy` for converting video files.
  - `ffmpeg` for audio and video processing.

### Installation

1. Install Python 3 from [python.org](https://www.python.org/).
2. Install the required Python libraries using `pip`:

   ```bash
   pip install yt-dlp moviepy
